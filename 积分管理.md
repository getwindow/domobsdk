#### 2.2 初始化积分墙SDK

**接口名称**

| 参数类型 | 参数名称 | 描述 |
| :--- | :--- | :--- |
| String | userid | APP的上下文 |
| SDKLoadListener | listener | 已请求过的任务数量 |
